# React + Vite

##Step 1: cd ai-assistant

##Step 2: Install Dependencies 
        npm install
        npm run dev

        The terminal will tell you the local address (e.g., http://localhost:5173/). Hold Ctrl and click that link to open the Adaptive AI Customer Service Assistant in your browser!

        npm install -D tailwindcss postcss autoprefixer
        npx tailwindcss init -p
